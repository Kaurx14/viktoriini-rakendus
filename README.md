# Statistikaamet viktoriin

## Kirjeldus

Tegemist on viktoriini rakendusega, kus kasutajale kuvatakase küsimusi. Küsimused on Eesti ja Eesti käiva statistika kohta. Veebilehe kujundus on tehtud jäljendama Statistikaameti ametlikku veebilehte.

### Täpsemalt

- Esitatakse 5 küsimust
- Igal küsimusel on 3 vastusevarianti, millest 1 on õige
- Pärast küsimusele vastamist, saab kasutaja kohe tagasisidet
- Viktoriini lõpus kuvatakse tulemused tabelina
- Lõpus kuvatakse ka kasutaja skoor ning isikupärastatud sõnum
- Leht toetab nii eesti kui ka inglise keelt


## Kasutatud tech stack
- React + Vite + TypeScript
- TailwindCSS
- Playwright (end-to-end testid)
